## Initial build — Wolfgang

Built Meowdoku Solver from scratch (session 1).

- `solver.js`: row-by-row backtracking CSP solver; checks column, color, and diagonal-adjacency constraints. Only checks row-1 for adjacency (proof: |row diff| > 1 cannot be adjacent).
- `app.js` + `style.css` + `index.html`: mobile-first UI. Drag-to-paint grid with touch + mouse events. 12-color palette. Grid scales 5×5 to 12×12 with cat emoji sized via `calc()` on a `--grid-size` CSS variable. Solution overlay with pop animation. Status bar with validation states.
- Docs: CLAUDE.md, session-index.md, TASKS.md, docs/ structure per `project-doc-structure.md`.
