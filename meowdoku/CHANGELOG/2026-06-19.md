## Screenshot import + hint system + GitHub Pages — Wolfgang

**image.js (new)**
Screenshot → grid pipeline: file picker → FileReader → auto-detect colorful-pixel bounding box (skips top 28% / bottom 15% UI chrome, aspect-ratio check) → preview overlay with red dashed box + faint grid lines → confirm or two-tap manual correction → 4-corner pixel sampling per cell (22%/78%, avoids X-marks) → greedy RGB clustering (threshold 50) → returns `{grid, colors, colorCount}`.

**app.js**
New state: `revealed` (how many rows' cats show), `customColors` (RGB strings from screenshot). `generateHintText(step)` builds rule-based explanation for each eliminated column (taken / color placed / diagonal / dead end). Hint button increments `revealed`; Solve All sets `revealed = size`. `cellColor(idx)` returns customColors when set, COLORS palette otherwise. Photo button wires to `openImagePicker`.

**style.css**
Modal styles, hint-box (yellow, pre-line), btn-hint (yellow). Modal overflow fix: `overflow-y: auto` + canvas `flex-shrink: 1`.

**index.html**
Added photo btn, hint btn, hint-box div, modal HTML, `<script src="image.js">`.

**.github/workflows/deploy.yml (new)**
Deploys `meowdoku/` folder to GitHub Pages on push to main. Live: https://woleywa.github.io/claude-playground/
