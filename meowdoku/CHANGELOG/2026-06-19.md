## Screenshot import + hint system + GitHub Pages — Wolfgang

**image.js (new)**
Screenshot → grid pipeline: file picker → FileReader → auto-detect colorful-pixel bounding box (skips top 28% / bottom 15% UI chrome, aspect-ratio check) → preview overlay with red dashed box + faint grid lines → confirm or two-tap manual correction → 4-corner pixel sampling per cell (22%/78%, avoids X-marks) → greedy RGB clustering (threshold 50) → returns `{grid, colors, colorCount}`.

**app.js**
New state: `revealed` (how many rows' cats show), `customColors` (RGB strings from screenshot). `generateHintText(step)` builds rule-based explanation for each eliminated column (taken / color placed / diagonal / dead end). Hint button increments `revealed`; Solve All sets `revealed = size`. `cellColor(idx)` returns customColors when set, COLORS palette otherwise. Photo button wires to `openImagePicker`.

**style.css**
Modal styles, hint-box (yellow, pre-line), btn-hint (yellow). Modal overflow fix: `overflow-y: auto` + canvas `flex-shrink: 1`.

**index.html**
Added photo btn, hint btn, hint-box div, modal HTML, `<script src="image.js">`.

**.github/workflows/deploy.yml (new)**
Deploys `meowdoku/` folder to GitHub Pages on push to main. Live: https://woleywa.github.io/claude-playground/

## Dark mode + detection overhaul + AI option — Wolfgang (Session 03)

**Dark mode (`style.css`)**
Static dark theme: `--bg #111`, light text, `#222` cells, white-transparent borders; buttons/slider/hint-box/status/modal all re-themed.

**Tile grid + X marks (`style.css`, `app.js`)**
Grid is now gap/padding rounded tiles with a drop-shadow (matches the original game). Revealed non-cat cells show `✕` (`.cell.excluded`, `xpop` animation) after solve/hint.

**Screenshot detection (`image.js`) — the big one**
- 5×5 median sampling per cell (was 4 points); `_medRGB` strips near-white X-mark pixels before the median.
- Clustering: threshold 50→60, post-merge of clusters <40 apart, then force-merge down to exactly N (`_mergePair`/`_closestPair`).
- **Square-clamp bounds:** action buttons below the grid inflated the auto-detect box ~23%; now clamped to the shorter side anchored top-left. Root-caused with a Python+Pillow harness on real screenshots.
- **Grid-size auto-detection** (`_detectGridSize`): counts tile bands (cols & rows) via the cream gaps; N when they agree in 5..12, else slider fallback. Used in extraction + preview. `handleImageResult` adopts the detected size.

**Clipboard paste (`image.js`, `app.js`, `index.html`)**
📋 button (`navigator.clipboard.read`) + global Ctrl/Cmd+V paste → `loadImageBlob`. `_loadFile`→`_loadBlob`, `_openFilePicker` extracted, file-picker fallback.

**AI detection (`image.js`, `app.js`, `index.html`, `style.css`)**
Optional 🤖 button (shown after import) re-runs detection on the last image via Gemini `gemini-flash-latest` (REST, JSON mode, 760px downscale, retry on 503/429). API key prompted once → `localStorage` only, never committed. Pixel sampling stays the default — Gemini tested slower and less accurate (merged rose↔pink, hallucinated a green) on the real screenshot.

**Layout (`index.html`, `style.css`)**
Size slider moved to a fixed bottom `.size-bar` (safe-area aware). `.btn-primary` text `#fff`→`#111` (Solve legibility; verify after deploy).

**Workflow:** direct-to-`main`, no feature branches for this project (per user).
