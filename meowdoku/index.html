Tactics guide
One per row / col / color Each row, column, and color region gets exactly one cat. Placing a cat immediately blocks the rest of that row, column, and color.
No touching Cats can't be adjacent — not even diagonally. All 8 surrounding cells of a cat are blocked.
Last valid cell If a row or color region has only one cell that isn't crossed out or otherwise blocked, the cat must go there.
Region locked to one row If all remaining cells of a color region fall in the same row, that row belongs to that color. Cross out other colors in that row.
Region locked to one column Same idea vertically. If a region only has cells in one column, cross out other colors in that column.
Pairs rule If two color regions together can only appear in the same two rows (or two columns), those rows are reserved for those two colors — eliminate other colors from them.
