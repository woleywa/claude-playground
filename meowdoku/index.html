Meowdoku Solver

Tactics guide
The ground rules Each row, column, and color gets exactly one cat. Cats can't touch — not even diagonally, so all 8 cells around a cat are blocked too.
🎯 Last Spot If a row, column, or color is down to exactly one open cell, the cat must go there.
📏 Line Lock If all of a color's open cells sit in one row (or column), that color owns the line — cross every other color out of it.
👥 Crowding If K colors can only fit in the same K rows (or columns), those lines are fully booked — cross every other color out of them. (K=2 is the classic "naked pair".)
🤏 Squeeze When a row, column, or color is down to just two or three candidates, any cell touching all of them is doomed — whichever one wins, it lands next door.
🤔 What-If Test a cell: pretend a cat sits there and follow every move that becomes forced. If some row, column, or color ends up with nowhere to go, that cell was impossible — cross it out.
